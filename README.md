# Proto UI

Proto UI is a UI library for low-fidelity prototypes.

## Component usage examples

### Badge
```html
<proto-badge>
  <proto-button type="text" size="larger">
    Click
  </proto-button>
</proto-badge>
```

### Image
```html
  <proto-image width="100px" height="100px" />
```

### Text
```html
  <proto-text width="100px" :height="3" :fontSize="24" />
```
